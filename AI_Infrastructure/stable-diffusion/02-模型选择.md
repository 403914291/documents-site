## 📦 什么是模型？

模型是 Stable Diffusion 的"大脑"，决定了生成图像的风格和质量。

**类比：**
- 🎨 基础模型 = 通用画家
- 🔧 微调模型 = 专业画家（擅长特定风格）
- 🎭 LoRA = 风格滤镜

## 📥 模型下载与安装

### 下载渠道

| 渠道 | 网址 | 特点 |
|------|------|------|
| **Civitai** | https://civitai.com/ | 最大模型站，有预览图 |
| **Hugging Face** | https://huggingface.co/ | 官方模型，稳定 |
| **LiblibAI** | https://www.liblib.ai/ | 国内镜像，速度快 |

### 下载步骤（以 Civitai 为例）

**1. 浏览模型**
```
访问 https://civitai.com/
搜索模型名称（如 Realistic Vision）
```

**2. 选择版本**
```
- 查看模型详情
- 选择 SDXL 或 SD1.5 版本
- 点击下载（.safetensors 格式）
```

**3. 安装模型**
```
方法 1：Stability Matrix 自动管理
  → 打开 Stability Matrix
  → 选择"模型管理"
  → 点击"导入模型"
  → 选择下载的.safetensors 文件

方法 2：手动复制
  → 复制模型文件到：
     stable-diffusion-webui/models/Stable-diffusion/
  → 重启 WebUI
  → 在模型列表中选择
```

## 🎚️ 模型融合技巧

### 什么是模型融合？

将多个模型按比例混合，创造新风格。

**公式：**
```
最终模型 = 模型 A × 比例 + 模型 B × (1-比例)
```

### 融合步骤（WebUI）

**1. 打开融合工具**
```
WebUI → Checkpoint Merger 标签
```

**2. 选择模型**
```
Primary model（主模型）: Realistic Vision
Secondary model（副模型）: DreamShaper
```

**3. 设置比例**
```
Multiplier (M) = 0.3
（主模型 30%，副模型 70%）
```

**4. 执行融合**
```
点击"Merge"按钮
等待完成（约 1-5 分钟）
输入新模型名称
```

### 融合比例参考

| 主模型 | 副模型 | 比例 | 效果 |
|--------|--------|------|------|
| 写实 | 梦幻 | 0.3 | 写实带梦幻感 |
| 写实 | 动漫 | 0.5 | 2.5D 风格 |
| 动漫 | 写实 | 0.4 | 半写实动漫 |
| 精细 | 快速 | 0.6 | 平衡质量速度 |

## 🎭 LoRA 模型使用

### 什么是 LoRA？

**LoRA（Low-Rank Adaptation）** 是小型风格适配器，只修改模型的部分参数。

**特点：**
- 📦 体积小（50-200MB）
- ⚡ 加载快
- 🎨 针对性强（特定风格/角色）
- 🔀 可组合使用

### LoRA 类型

| 类型 | 作用 | 示例 |
|------|------|------|
| **风格 LoRA** | 添加艺术风格 | 水墨画、油画、赛博朋克 |
| **角色 LoRA** | 生成特定角色 | 动漫角色、名人 |
| **服装 LoRA** | 添加特定服装 | 汉服、校服、西装 |
| **姿势 LoRA** | 控制人物姿势 | 坐姿、站姿、动作 |
| **画质 LoRA** | 提升图像质量 | 细节增强、光影优化 |

### LoRA 使用方法

**1. 下载 LoRA**
```
访问 Civitai → 搜索 LoRA → 下载.safetensors 文件
```

**2. 安装 LoRA**
```
复制到：stable-diffusion-webui/models/Lora/
重启 WebUI
```

**3. 在提示词中调用**
```
正向提示词添加：
<lora:style_watercolor:0.8>
（格式：<lora:文件名：权重>）
```

**4. 调整权重**
```
权重范围：0.1 - 1.5
推荐：0.6 - 0.8
过高会导致画面崩坏
```

### 多 LoRA 组合

**语法：**
```
<lora:lora1:0.6> <lora:lora2:0.4>
```

**示例：**
```
<lora:chinese_dress:0.7> <lora:ancient_building:0.5>
（汉服 + 古建筑组合）
```

## 📝 常见问题

### Q1: 模型太多怎么管理？

**解决方案：**
1. 使用 Stability Matrix 的模型管理功能
2. 建立文件夹分类（人像/风景/动漫）
3. 删除长期不用的模型
4. 使用模型融合代替多个模型

### Q2: 为什么别人的图比我好看？

**可能原因：**
1. 使用了更合适的模型
2. 提示词更精准
3. 参数调优更好
4. 使用了 LoRA 或 ControlNet

### Q3: 模型可以商用吗？

**答案：** 大部分可以，但需查看具体许可证
- CreativeML Open RAIL-M：可商用
- 部分模型有使用限制
- 建议在 Civitai 查看模型页面说明

### Q4: 如何备份模型？

**建议：**
1. 外接硬盘存储（模型文件大）
2. 记录模型名称和下载链接
3. 使用云存储（Google Drive/OneDrive）
4. 导出 WebUI 配置
