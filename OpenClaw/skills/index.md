# OpenClaw Skills 技能系统

> OpenClaw 技能系统完全指南 - 扩展你的 AI 助手能力

---

## 📚 技能系统概述

OpenClaw 使用**技能系统**来扩展功能。技能是可复用的功能模块，可以通过 ClawHub CLI 从 [clawhub.com](https://clawhub.com) 安装。

### 技能能做什么

- 🖼️ **图像生成** - AI 绘图、图像编辑
- 🌤️ **天气查询** - 实时天气和预报
- 📝 **GitHub 操作** - Issues、PR、代码审查
- 🎬 **视频处理** - 视频帧提取和剪辑
- 📱 **社交媒体** - X (Twitter) API 操作
- 🔍 **技能发现** - 帮助发现和安装技能

## 🚀 快速开始

### 1. 安装 ClawHub CLI

```bash
npm install -g clawhub
```

### 2. 搜索技能

```bash
# 搜索关键词
clawhub search "image generation"
clawhub search "weather"
```

### 3. 安装技能

```bash
# 安装最新版本
clawhub install baoyu-image-gen
```

### 4. 验证安装

```bash
# 列出已安装的技能
clawhub list
```
